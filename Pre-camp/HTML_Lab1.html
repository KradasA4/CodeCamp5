1. attribute
tag Name attribute การใช้งาน
<h> ไม่มี attribute ที่ซัพพอร์ตใน html5
<p> ไม่มี attribute ที่ซัพพอร์ตใน html5
<div> ไม่มี attribute ที่ซัพพอร์ตใน html5
<img> alt กำหนดข้อความที่แสดงแทน เมื่อหน้าเว็บไม่แสดงรูปภาพ
crossorigin อนุญาตให้เว็บไซต์อื่นดึงภาพไปใช้งานได้
height กำหนดความสูงของรูปภาพ
ismap เชื่อมรูปภาพเข้ากับลิงก์แบบ server-side ข้อมูลการคลิกจะถูกส่งไปยัง server
longdesc บรรยายรายละเอียดของรุปภาพอย่างยาวกว่า attribute alt
sizes กำหนดขนาดของรูปภาพ
src กำหนด url ที่มาของรูปภาพ
srcset กำหนด url ที่มาของรูปภาพในสถานการณ์อื่น เช่น หน้าจอมีขนาดใหญ่ขึ้นหรือเล็กลง
usemap เชื่อมรูปภาพเข้ากับลิงก์แบบ client-side สามารถกำหนดพื้นที่ที่คลิกได้ ไม่จำเป็นต้องคลิกได้ทั้งภาพ เช่น กำหนดพื้นที่ที่คลิกได้เป็นวงกลมในภาพ
width กำหนดความกว้างของรูปภาพ
<a> download ให้ดาวน์โหลดไฟล์ตามลิงก์ที่กำหนดเมื่อคลิกูปภาพ
href ระบุเว็บเพจที่จะลิงก์ไปเมื่อรูปภาพถูกคลิก
hreflang ระบุภาษาของหน้าเว็บที่จะลิงก์ไป
media กำหนดค่าการแสดงผลหน้าที่จะลิงก์ไปให้เหมาะสำหรับอุปกรณ์ชนิดใดชนิดหนึ่งใดโดยเฉพาะ
ping ส่ง ping เมื่อลิงกืถุกคลิก ใช้สำหรับนับจำนวนการคลิก
referrerpolicy กำหนดว่าจะส่งหรือไม่ส่ง referrer ไปยังลิงก์ปลายทาง ว่าการลิงก์นี้มาจากเว็บไหน
rel ระบุความสัมพันธ์ระหว่างหน้าเว็บปัจจุบันกับหน้าที่ลิงก์ไป
target กำหนดว่าจะเปิดลิงก์นี้ที่ไหน เช่น ในหน้าต่างใหม่
type ระบุ media type ของหน้าที่ลิงก์ไป เช่น เป็นเว็บ html หรือ เป็นไฟล์ script
<ul> ไม่มี attribute ที่ซัพพอร์ตใน html5
<ol> reversed กำหนดให้ ordered list เรียงลำดับแบบนับย้อนจากมากไปน้อย
start กำหนดค่าว่าจะให้ ordered list เริ่มนับจากลำดับที่เท่าไร
type กำหนดประเภทของ ordered list ลิสต์ เช่น นับเป็นตัวเลขอาราบิก ลขโรมัน หรืออักษรอัลฟาเบต
<table> ไม่มี attribute ที่ซัพพอร์ตใน html5
<form> accept-charset กำหนดฟอนต์และค่าของรูปแบบอักษรสำหรับข้อความในฟอร์ม
action กำหนดว่าให้ส่งข้อมูลในฟอร์มไปยัง url ใด หลังผู้ใช้กด submit
autocomplete กำหนดว่าจะอนุญาตให้เบราเซอร์ autocomplete ฟอร์มหรือไม่
enctype กำหนดว่าจะ encode ข้อมูลในฟอร์มอย่างไรก่อนส่งไปยังเซิร์ฟเวอร์
method กำหนด html method ที่เบราเซอร์ใช้รับส่งข้อมูลกับเซิร์ฟเวอร์
name ระบุชื่อฟอร์ม
novalidate ปิดการตรวจสอบค่าฟอร์มทั้งหมด รวมถึงการตรวจสอบด้วย attribute อื่นด้วย
target กำหนดว่าจะเปิดหน้าเว็บหลัง submit ในหน้าต่างใด
<button> autofocus เมื่อหน้าเว็บโหลด ให้เน้นการแสดงผลที่ปุ่มซึ่งมี attribute นี้
disabled กำหนดให้ปุ่มนี้คลิกไม่ได้
form ระบุว่าปุ่มนี้สัมพันธ์กับฟอร์มไหน แม้แท็ก button จะอยู่นอก form
formaction กำหนดว่าจะส่งข้อมูลในฟอร์มไปยัง url ใด ใช้สำหรับ type="submit" เท่านั้น
formenctype กำหนดว่าจะ encode ข้อมูลในฟอร์มอย่างไรก่อนส่งไปยังเซิร์ฟเวอร์ ใช้สำหรับ type="submit" เท่านั้น
formmethod กำหนด html method ที่เบราเซอร์ใช้รับส่งข้อมูลกับเซิร์ฟเวอร์เมื่อ submit ด้วยปุ่ม ใช้สำหรับ type="submit" เท่านั้น
formnovalidate ปิดการตรวจสอบค่าฟอร์มทั้งหมด รวมถึงการตรวจสอบด้วย attribute อื่นด้วย ใช้สำหรับ type="submit" เท่านั้น
formtarget ระบุว่าจะเปิด url หน้าเว็บหลัง submit ด้วยปุ่มผ่านหน้าต่างใด ใช้สำหรับ type="submit" เท่านั้น
name ระบุชื่อของปุ่ม
type กำหนด type ของปุ่ม
value ระบุข้อความที่แสดงบนปุ่ม

2. differentiation tag
ความเหมือน ความต่าง
<hr> & <br> เป็น empty tag ที่แสดงผลโดยขึ้นบรรทัดใหม่เหมือนกัน hr (horizontal rule) เป็น block-level element แสดผงลเป็นขีดแนวนอนแบ่ง section
br (break) เป็น inline element ใช้กำหนดให้แสดงข้อความหลังแท็กนี้ในบรรทัดใหม่
<ul> & <ol> ใช้เพื่อแสดงผลข้อมูลในแท็ก li เป็นลิสต์รายการเหมือนกัน ul จะแสดงผลแท็ก li เป็น bullet ไม่กำกับลำดับของข้อมูล
ol จะแสดงผลแท็ก li โดยกำกับลำดับข้อมูลด้วย เช่น ลำดับตัวเลขหรืออักษรอัลฟาเบต
<select> & <checkbox> & <radio button> ใช้แสดงผลให้ผู้ใช้งานเลือกตอบจากตัวเลือกที่กำหนดให้เหมือนกัน select เป็นรายการแบบ drop-down เลือกตอบได้แค่ตัวเลือกเดียว จำเป็นต้องเลือกตอบ
checkbox เป็นรายการแบบลิสต์ เลือกตอบได้หลายตัวเลือก หรือไม่เลือกเลยก็ได้
radio button เป็นรายการแบบลิสต์ เลือกตอบได้แค่ตัวเลือกเดียว จำเป็นต้องเลือกตอบ
<div> & <span> ใช้กำหนดแบ่ง section ของข้อมูลในหน้าเพจเหมือนกัน div เป็น block-level element
span เป็น inline element
<script> & <style> ใช้กำหนดคุณสมบัติและการแสดงผลของหน้าเว็บ ด้วยการกำหนดภายใน html เหมือนกัน script ใช้จัดการการปฏิสัมพันธ์กับเว็บและแสดงผล action ต่างๆ ด้วยภาษาสคริปต์อย่าง JavaScript หรือ TypeScript
style ใช้ปรับแต่งการแสดงผลในหน้าเว็บด้วย CSS

3. input type
input type ลักษณะการทำงาน
button สร้างปุ่มเพื่อรับข้อมูลการคลิก
checkbox เป็นตัวเลือกแบบลิสต์ ผู้ใช้งานสามารถเลือกตอบได้หลายตัวเลือกหรือไม่เลือกเลยก็ได้
color ให้ผู้ใช้งานเลือกสีจากแถบสี
date ให้ผู้ใช้เลือกเดือน วัน และปีที่ต้องการระบุ
datetime-local รับข้อมูลเดือน วัน ปี และเวลา
email รับข้อมูลประเภท email เท่านั้น จำเป็นต้องมีเครื่องหมาย @
file รับและอัพโหลดไฟล์ที่ผู้ใช้เลือก
hidden ไม่แสดงการรับ input ให้ผู้ใช้งานเห็น แต่หากกำหนดให้ส่งข้อมูล ข้อมูลนั้นจะยังถูกส่งไปยังเซิร์ฟเวอร์
image แดงรูปภาพ และรับข้อมูลตำแหน่งที่รูปภาพถูกคลิก
month รับข้อมูลเดือนและปีจากผู้ใช้งาน
number รับข้อมูลประเภทตัวเลข กำหนดได้ว่าให้เลือกตัวเลขใน range ไหน
password รับข้อมูลโดยซ่อนข้อมูลไม่ให้เห็นจากหน้าจอ และไม่สามารถ copy ได้
radio กำหนดหลายตัวเลือกให้ผู้ใช้เลือกได้เพียงตัวเลือกเดียว โดยจำเป็นต้องเลือก
range รับข้อมูล range ของค่าตัวเลข โดยให้เลื่อนเลือกจุดใดจุดหนึ่งบนระนาบ
reset รีเซ็ตค่าที่ถูกกรอกให้กลับไปเป็นค่าเริ่มต้นที่ตั้งไว้
search ใช้สำหรับรับข้อมูลที่ต้องการค้นหา แสดงผลแบบเดียวกับ text field
submit ้ใช้ส่งข้อมูลไปยังที่ที่ฟอร์มกำหนดไว้
tel รับข้อมูลหมายเลขโทรศัพท์ สามารถกำหนดแพทเทิร์นตัวเลขที่ต้องการให้กรอกได้
text รับข้อมูลประเภทข้อความทั่วไป 1 บรรทัด
time รับข้อมูลเวลาที่ผู้ใช้ระบุ
url รับข้อมูลประเภท url โดยจำเป็นต้องกรอก protocal ด้วย
week รับข้อมูลสัปดาห์และปีจากผู้ใช้งาน โดยเลือกสัปดาห์เป็นตัวเลข 1 ถึง 52
